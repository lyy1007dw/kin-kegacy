# Kin-Legacy 家谱管理系统

家谱管理系统，支持家族成员信息管理、家谱树状展示、审批流程等功能。

## 项目结构

```
kin-legacy/
├── kin-legacy-mp/      # 微信小程序端 (uni-app)
├── kin-legacy-admin/   # 后台管理系统 (Vue3 + TypeScript)
└── 参考文档/           # 项目需求文档
```

## 功能模块

- 用户管理：支持超级管理员、家谱管理员、普通用户三种角色
- 家谱管理：家谱创建、成员管理、权限设置
- 成员管理：成员信息录入、关系树展示
- 审批管理：成员加入审批流程

## 技术栈

### 小程序端
- uni-app
- Vue 3
- Vuex

### 后台管理
- Vue 3
- TypeScript
- Vite
